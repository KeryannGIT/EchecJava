# Jeu d'échecs

## Description
Ce projet est un jeu d'échecs développé en Java. Pour lancer le jeu, vous devez compiler les fichiers sources et ensuite exécuter le programme principal.

## Prérequis
- Java JDK installé
- Terminal / Invite de commandes

## Compilation et lancement

### Sur Linux
1. Ouvrez le terminal dans le répertoire du projet.
2. Exécutez le fichier `start.sh` :
   ```shell
   start.sh

### Sur Windows
1. Ouvrez l'invite de commandes dans le répertoire du projet.
2. Exécutez le fichier `start.bat` :
   ```bat
   start.bat
